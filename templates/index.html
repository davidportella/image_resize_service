github.com/flurischt/image_resize_service

setup

  • configure SOURCE_DIR and RESIZED_DIR in production.cfg
  • add a project and some image dimensions to production.cfg
  • make sure the directories SOURCE_DIR/projectname and RESIZED_DIR/projectname exist
  • put some images to SOURCE_DIR/projectname/
  • access them using /img/project_name/image_name@dimension.image_extension
example
  • demo_project/welcome@small.jpg
  • demo_project/welcome@medium.jpg
  • demo_project/welcome@large.jpg
  • demo_project/welcome@fullsize.jpg

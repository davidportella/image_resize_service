setup

  • choose a STORAGE in production.cfg
  • configure SOURCE_DIR and RESIZED_DIR if STORAGE=FILESYSTEM
  • upload some images or add them to SOURCE_DIR/project/ (if storage=FILESYSTEM)
  • access them using /img/project_name/image_name@dimension.image_extension to get a resized image
  • or access them using /img/project_name/image_name.extension to get the full image
